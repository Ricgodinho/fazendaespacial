# Fazenda Espacial

Jogo indie de fazenda espacial, para Steam (PC).

**Status:** ideação inicial — em debate.

## Pitch (rascunho)

Um jogo de gestão de fazenda com loop híbrido (ativo + passivo/idle), onde o jogador
expande operações agrícolas por um sistema solar com múltiplos planetas. Gráficos
mais atraentes que uma referência estilo pixel/idle, mas mantendo leveza de performance.
